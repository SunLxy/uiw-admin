设置页面标题
---

## Installation

```bash
npm i @uiw-admin/document-title --save
```

## Basic Usage

```jsx
import React from 'react';
import DocumentTitle from '@uiw-admin/document-title';

<DocumentTitle title="首页">
  <h1>Home, sweet home.</h1>
</DocumentTitle>
```
